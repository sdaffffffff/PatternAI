# Análisis Competitivo

## Competidores Principales

### CLO3D
- **Qué hace:** Software de diseño y simulación 3D de prendas, estándar de la industria
- **Precio:** ~$1,500-2,500/año
- **Fortalezas:** Simulación de física de tela muy precisa, integraciones industriales, base de usuarios consolidada
- **Debilidades:** Curva de aprendizaje muy alta, orientado a patronistas técnicos, no accesible para diseñadores sin formación
- **Novedad relevante:** Lanzaron "Boceto sobre avatar 2026.0" — permite dibujar sobre un avatar para generar prendas. **Señal de que van hacia nuestro mercado.**
- **Nuestra posición:** No competimos en simulación 3D de física — competimos en accesibilidad y simplicidad para diseñadores no técnicos

### Lectra / Gerber Technology / Optitex
- **Qué hacen:** Software CAD/CAM industrial para patronaje, grading y corte
- **Precio:** Enterprise, decenas de miles de euros/año
- **Fortalezas:** Integración total con fábricas, estándar en grandes marcas
- **Debilidades:** Solo para empresas grandes, nada accesible, no tienen IA conversacional
- **Nuestra posición:** No competimos aquí en MVP — son el destino al que exportamos (DXF compatible)

### Meshy
- **Qué hace:** Generación de modelos 3D a partir de imágenes o texto
- **Relevancia para nosotros:** Candidato para la capa de visualización 3D en Fase 2
- **Limitación:** Output de ropa en 3D aún irregular — funciona mejor para objetos rígidos
- **Uso previsto:** API para visualización en Fase 2, no para patronaje

### Wild Alley / Otros AI Fashion Tools
- **Qué hacen:** Generación de looks, styling, recomendaciones de moda via IA
- **Diferencia clave:** Son herramientas de inspiración y marketing, no de producción técnica
- **Nuestra posición:** No somos una herramienta creativa — somos infraestructura de producción

## Ventana de Oportunidad

CLO3D acaba de lanzar su feature de boceto sobre avatar. Esto confirma que el mercado va en esta dirección, pero también significa que la ventana para establecerse antes de que lo hagan bien es limitada.

**Ventaja competitiva que debemos construir rápido:**
1. UX radicalmente más simple — diseñadores no técnicos como primer ciudadano
2. Precio accesible — el CLO3D del indie designer
3. Flujo conversacional en español e inglés desde el inicio
4. Tabla de medidas por marca — personalización que CLO3D no ofrece fácilmente
